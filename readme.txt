== Description ==

Met de Wordpress Couverts plugin van [Tussendoor](https://tussendoor.nl) integreer je Couverts binnen jouw WordPress site volledig in je eigen huisstijl.

Stop met het gebruik van het Couverts iFrame, maar integreer Couverts zoals het hoort. Via diverse instellingen en widgets plaats je de Couverts boekingkalender waar je maar wilt, zoals je maar wilt. Geen beperkingen in opmaak en kleur, maar volledig passend bij de uitstraling van jouw website.

Als restaurant houder wil je je bezoekers natuurlijk zo eenvoudig mogelijk laten reserveren. Couverts heeft daar de ideale oplossing voor, alleen kun je dit via hun iFrame onmogelijk mooi integreren in je eigen website. Daar is deze Couverts plugin voor!

Met de WordPress Couverts plugin, integreer je Couverts 100% in je eigen huisstijl in je eigen website. Hierdoor heb een je volledig eigen uitstraling, kun je schalen voor mobiel en tablet (waarop men vaak reserveert) terwijl je toch 100% Couverts gebruikt. Onmisbaar voor ieder restaurant!

1. Één centrale reservering-module;
2. Volledig op basis van de Couverts API;
3. Automatische updates via de WordPress repo;
4. Presteert optimaal op tablet en smartphone!

== Installation ==

1. Upload de plugin naar je WordPress site. Dit kan handmatig (via FTP) en via het WordPress dashboard.
2. Activeer de plugin onder de plugin pagina
3. Voer de API gegevens van Couverts in
4. Plaats de shortcode [couvertsForm] op een pagina om het formulier in te laden
